# Buildify

An AI-powered app builder that lets you generate full applications, UI layouts, and dashboards through a chat interface. Built on the T3 Stack with Next.js 15, it integrates the V0 SDK for code generation and Razorpay for subscription-based monetization.

## Features

- **AI App Builder** - Generate full applications, dashboards, landing pages, and UI layouts through guided chat-based AI generation powered by the V0 SDK
- **AI Chat** - Ask questions, debug code, brainstorm ideas, and iterate quickly with real-time AI responses
- **Buildify Studio** - Visual project workspace for managing and iterating on generated applications
- **AI Resume Builder** - Fill in your details and get a professional LaTeX resume generated by AI (via OpenRouter), with follow-up editing and PDF compilation
- **Credit System** - Dual credit model with subscription credits (monthly) and additional credits (never expire). New chat costs 20 credits, follow-up costs 30
- **Subscription Plans** - Free tier (200 credits), Starter, Pro, and Enterprise plans with Razorpay payment integration
- **Multi-currency Pricing** - Localized pricing with automatic currency detection
- **Auth** - Email/password and GitHub OAuth via Better Auth
- **Rate Limiting** - 50 messages/day for authenticated users, 3/day for anonymous

## Getting Started

### Prerequisites

- [Bun](https://bun.sh) (v1.0+)
- PostgreSQL database
- API keys for: V0 SDK, Razorpay, OpenRouter, SMTP provider

### Installation

```bash
git clone https://github.com/khalidkhnz/techno-builder.git
cd techno-builder
bun install
```

### Environment Variables

Copy `.env.example` to `.env` and fill in the required values:

```env
# Core
DATABASE_URL="postgresql://..."
BETTER_AUTH_SECRET="your-secret"
NEXT_PUBLIC_APP_URL="http://localhost:3000"

# AI
V0_API_KEY="your-v0-api-key"
OPENROUTER_API_KEY="your-openrouter-key"

# Payments
RAZORPAY_KEY_ID="your-razorpay-key"
RAZORPAY_KEY_SECRET="your-razorpay-secret"
NEXT_PUBLIC_RAZORPAY_KEY_ID="your-razorpay-key"

# Email (SMTP)
SMTP_HOST="smtp.example.com"
SMTP_PORT=587
SMTP_USER="your-email"
SMTP_PASS="your-password"

# OAuth (optional)
GITHUB_CLIENT_ID="..."
GITHUB_CLIENT_SECRET="..."
```

Skip env validation during build with `SKIP_ENV_VALIDATION=1`.

### Database Setup

```bash
bun run db:push       # Push schema to database
# or
bun run db:generate   # Generate migration files
bun run db:migrate    # Run migrations
```

### Development

```bash
bun run dev           # Start dev server
```

Open [http://localhost:3000](http://localhost:3000).

## Architecture

**API Layer**: Elysia app exposed through a Next.js catch-all route at `/api/[[...slugs]]`. The Eden treaty client provides end-to-end type safety from server to client.

**Credit System**: Subscription credits reset each billing cycle. Additional purchased credits never expire. Both are deducted before API calls.

**Database**: All tables prefixed with `Buildify_` via `pgTableCreator`, except auth tables which use plain `pgTable`.
